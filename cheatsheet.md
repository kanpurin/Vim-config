# Vim Cheatsheet

よく使うコマンドだけを短く確認する表です。

## 基本方針

| 操作 | 意味 |
| --- | --- |
| `v` -> 範囲選択 -> `y` | 文字単位で選んでコピー |
| `v` -> 範囲選択 -> `d` | 文字単位で選んで削除 |
| `V` -> 範囲選択 -> `y` | 行単位で選んでコピー |
| `V` -> 範囲選択 -> `d` | 行単位で選んで削除 |

## ビジュアルモード

| コマンド | 意味 |
| --- | --- |
| `v` | 文字単位のビジュアルモード |
| `V` | 行単位のビジュアルモード |
| `y` | 選択範囲をコピー |
| `d` | 選択範囲を削除 |

## 移動

| コマンド | 意味 |
| --- | --- |
| `w` | 次の単語へ移動 |
| `b` | 前の単語へ移動 |
| `gg` | ファイル先頭へ移動 |
| `G` | ファイル末尾へ移動 |
| `%` | 対応する括弧へ移動 |

## マーク

| コマンド | 意味 |
| --- | --- |
| `ma` | 現在位置に `a` というマークを付ける |
| `` `a `` | マーク `a` の正確な位置へ移動 |
| `` `` `` | 直前にいた正確な位置へ戻る |

## ジャンプ履歴

| コマンド | 意味 |
| --- | --- |
| `Ctrl-o` | 前のジャンプ位置へ戻る |
| `Ctrl-i` | 次のジャンプ位置へ進む |

## 編集

| コマンド | 意味 |
| --- | --- |
| `dd` | 1行削除 |
| `yy` | 1行コピー |
| `p` | 貼り付け |
| `diw` | 単語を削除 |
| `yiw` | 単語をコピー |

## 戻す・やり直す

| コマンド | 意味 |
| --- | --- |
| `.` | 直前の変更を繰り返す |
| `u` | 元に戻す |
| `Ctrl-r` | やり直し |

## 検索・置換

| コマンド | 意味 |
| --- | --- |
| `*` | カーソル下の単語を検索 |
| `:%s/old/new/g` | ファイル全体で置換 |
| `:%s/old/new/gc` | 確認しながらファイル全体で置換 |

## ファイル・分割・タブ

| コマンド | 意味 |
| --- | --- |
| `:e ファイル名` | 別ファイルを開く |
| `:sp ファイル名` | 横分割で開く |
| `:vsp ファイル名` | 縦分割で開く |
| `Ctrl-w h` | 左の分割ウィンドウへ移動 |
| `Ctrl-w j` | 下の分割ウィンドウへ移動 |
| `Ctrl-w k` | 上の分割ウィンドウへ移動 |
| `Ctrl-w l` | 右の分割ウィンドウへ移動 |
| `Ctrl-w c` | 今の分割ウィンドウを閉じる |
| `Ctrl-w o` | 今の分割ウィンドウだけ残す |
| `Ctrl-w H` | 今の分割ウィンドウを一番左へ移動 |
| `Ctrl-w J` | 今の分割ウィンドウを一番下へ移動 |
| `Ctrl-w K` | 今の分割ウィンドウを一番上へ移動 |
| `Ctrl-w L` | 今の分割ウィンドウを一番右へ移動 |
| `:tabnew ファイル名` | 新しいタブで開く |
| `gt` | 次のタブへ移動 |
| `gT` | 前のタブへ移動 |
| `:tabclose` | 今のタブを閉じる |
| `:tabonly` | 今のタブ以外を閉じる |
