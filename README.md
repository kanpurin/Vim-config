# Vim Config Notes

Vim のコマンドメモと、自分用の `.vimrc` を育てるためのリポジトリです。

## 方針

基本すぎる操作は省き、実際に編集効率が上がるコマンドを中心に記録します。

特に、範囲を選んでから操作する流れを基本にします。

1. `v` または `V` でビジュアルモードに入る
2. 範囲を選択する
3. `y` でコピー、`d` で削除する

## ファイル

- `commands.md`: 整理済みのコマンドメモ
- `practice.md`: 覚えたて、練習中のコマンド
- `cheatsheet.md`: よく使うものだけを短く確認する表
- `.vimrc`: 自分用の Vim 設定

## メモの基準

各コマンドは、できるだけ次の3つをセットで書きます。

- 何をするか
- どう使うか
- どんな場面で使うか
